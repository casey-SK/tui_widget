The widgets used to work as a standalone widget and on a combinations uses of multiple widgets navigate via `[tab]` button. 

Refers to tui_app.nim for example.

Widgets:
- input box (y)

- display panel (y)

- button

- list view (wip)

- select

- table

- checkbox (y)
